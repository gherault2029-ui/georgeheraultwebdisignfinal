**Web Design Final - Part I
Correcting HTML and CSS errors**

You have been given a web page that contains several HTML and CSS errors. 
Your job is to find and fix the errors so that the page displays correctly.

You should correct errors related to:

- HTML structure
- Missing or incorrect tags
- CSS selectors
- Class and ID names
- Image display
- Navigation links
- Layout and spacing
- JavaScript is not required for this section

**Directions**
1. Copy the HTML into an index.html file.
2. Copy the CSS into a style.css file.
3. Open the page in a browser.
4. Identify and fix as many errors as you can.
5. When finished, the page should look organized, styled, and readable.
6. Submit both corrected files.

**Point Value: 25 points**
- HTML structure and syntax: **8 points**
- CSS syntax: **6 points**
- Correct use of classes, IDs, and selectors: **6 points**
- Layout and visual improvement: **3 points**
- Neatness and readable code: **2 points**
